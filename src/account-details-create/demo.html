Account details create

This component allows you to create a recipient for a given currency.

<account-details-create
  profile-id="..."
  currency="'{{ $ctrl.currency }}'"
  locale="'{{ $ctrl.locale }}'"
  quote-id="..."
  on-change="myChangeHandler(model)"
  on-success="mySuccessHandler()"
  on-failure="myFailureHandler()"
  save-button-label="'Save'"
></account-details-create>
profileID
Which profile will own this account.
currency
3 digit currency code, used to determine the requirements for the new account.
locale
(Optional) Which language should we show? Default: en-GB
quoteID
(Optional) If we're creating a new recipient for a quote, provide the quote id in addition to the currency. Occasionally the requirements may change based on other aspects of the quote (e.g. amount).
onChange(model)
(Optional) Triggered every time the model changes.
onSaveSuccess
(Optional) Triggered when an account is saved successully.
onSaveFailure
(Optional) Triggered when an account save fails. Error messaging is handled internally.
saveButtonLabel
Localised label for the save button.

Internal model

{{ $ctrl.model | json }}
